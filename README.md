# 多 Agent 协同运营自动化系统

一个面向销售获客场景的本地演示项目，用多 Agent 协作方式把线索筛选、客户研究、分层运营、个性化触达和自动跟进串成一条完整流程。

## 项目亮点

- 多 Agent 协同：不是单次生成，而是 6 个角色分工协作
- 本地可运行：基于 Python 标准库实现，不依赖额外安装
- 自带可视化控制台：适合演示、答辩和作品展示
- 自带样本数据：内置 12 条销售线索，开箱即用
- 本地结果落库：自动保存运行记录，方便复盘

## Agent 架构

系统包含以下 6 个 Agent：

1. `Lead Scout Agent`
   负责筛选符合 ICP 的目标客户
2. `Research Agent`
   负责分析客户痛点、触发事件和成交优先级
3. `Segment Agent`
   负责客户分层和运营策略设计
4. `Outreach Agent`
   负责生成个性化邮件、LinkedIn 文案和电话开场
5. `Sequence Agent`
   负责制定自动跟进节奏
6. `Ops Manager Agent`
   负责汇总执行剧本和周度 KPI

## 功能概览

- 输入产品名称、目标行业、目标角色、目标市场等参数
- 自动筛选高匹配线索
- 自动判断优先级和客群类型
- 自动生成触达话术和推荐渠道
- 自动输出 5 步跟进序列
- 自动记录历史运行结果
